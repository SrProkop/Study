Программа десиреализует информацию из файла typeXml.txt,
после чего создаёт объект с хранящимеся данными из файла,
после чего записывает объект в новый файл typeJson.txt

typeXml.txt - текстовый файл, хранящий информацию в похожем на xml виде
typeJson.txt - текстовой файл, который является результатом выполнения программы
и записывает объект в похожем на json виде
